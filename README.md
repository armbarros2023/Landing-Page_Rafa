# Landing Page Rafa

Projeto Vite para a landing page da Imersão Igreja Acolhedora.

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build para publicação

```bash
npm run build
npm run preview
```

## Publicação

Os arquivos de imagem usados pela landing ficam em `public/Fotos Landing Page Rafa`, para que o build do Vite publique os assets corretamente.
